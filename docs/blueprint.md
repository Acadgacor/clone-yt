# **App Name**: CineView

## Core Features:

- Video Embedding: Embed a YouTube video using an iframe from the provided URL.
- Video Playback: Play the embedded video directly on the website.
- AI-Powered Recommendation Tool: Uses an AI model to suggest similar or related videos based on the currently playing video; incorporates genre analysis and user ratings to decide which videos to suggest.
- User Controls: Provide standard video playback controls (play, pause, volume, fullscreen).

## Style Guidelines:

- Primary color: Dark purple (#673AB7) to convey a cinematic feel.
- Background color: Very dark gray (#121212) to provide contrast and enhance the viewing experience.
- Accent color: Bright orange (#FF9800) for interactive elements and highlights.
- Body and headline font: 'Inter', a grotesque-style sans-serif font.
- Use clean, minimalist icons for video controls and navigation.
- The video player should take center stage, with recommendations and other features arranged around it in a non-obtrusive way.
- Use subtle animations for loading and transitioning between videos.
